# MP3 Tag Reader

A command-line application written in C to read and edit MP3 metadata (ID3v2 tags). The project demonstrates binary file handling, file parsing, and manipulation of MP3 tag information.

## Features

- View MP3 metadata
- Edit song title
- Edit artist name
- Edit album name
- Edit year
- Edit genre
- Binary file parsing
- ID3v2 tag processing
- Command-line interface

## Technologies Used

- C Programming
- File Handling
- Binary Files
- Structures
- Pointers
- Linux (GCC)

## Project Structure

```
main.c
viewMp3.c
edit.c
functions.h
```

## Build

```bash
gcc *.c 
```

## Run

```bash
./a.out sample.mp3
```

## Learning Outcomes

- Binary file manipulation
- MP3 ID3 tag format
- Command-line argument processing
- Dynamic memory usage
- File pointer operations
